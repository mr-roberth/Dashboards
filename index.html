Información del Dashboard

Este dashboard de Power BI proporciona una vista completa del rendimiento de la fábrica, incluyendo métricas de producción, eficiencia, calidad y mantenimiento. Los datos se actualizan periódicamente para garantizar la precisión de la información.

Interacción

Haga clic en los diferentes gráficos y elementos para filtrar la información. Use la función de zoom para examinar áreas específicas.

Filtrado de Datos

Utilice los filtros en la parte superior del dashboard para seleccionar períodos de tiempo, líneas de producción o turnos específicos.

Exportación

Para exportar datos, haga clic en el icono de exportación en la esquina superior derecha de cualquier visualización.
